# Project Dashboard

A comprehensive Project Dashboard designed for tracking project status, milestones, and high-level KPIs.

## Features
- **KPI Monitoring**: Instant visibility into projects on track, performance critical, and behind schedule.
- **Decision Timeline**: Visual representation of upcoming milestones and decision points across different projects.
- **Detailed Project List**: Grouped table view with status tracking, RAG indicators, and management priority alignment.
- **Interactive Controls**: Table-level sorting, searching, and view configuration.
